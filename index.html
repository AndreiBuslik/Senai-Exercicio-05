Aplicando estilos CSS através da tag <style><\style>

Sobre estilos

O uso de CSS pode ser feito tanto com a utlização da tag <link>, quando queremos que a manipulação dos estilos da página estejam em um arquivo separado, como a tag <style><\style>.

A tag <style><\style> mantém a manipulação das modificações de estilo no mesmo arquivo do arquivo de hipertexto (.html). Serão demonstrados alguns usos de CSS utilizando a tag <style><\style>. Não discutiremos sobre se devemos usar a tag <style><\style> ou fazer um arquivo em seprado para o CSS e utilizar a tag <link>. Nosso objetivo neste momento é aprender algumas propriedades do CSS e o que ocorre com a página quando as utlizamos.

Fontes, cores e outras manipulações de texto

Podemos utilizar o CSS para modificar o texto da página. Uma das modificações possíveis é na fonte do texto.

Mudando a fonte podemos deixa um texto mais legível ou agradável. Perceba que o parágrafo anterior possui uma fonte diferente deste, enquanto no primeira temos uma fonte da família Verdana nesta seção temos uma fonte com estilo de letra cursiva (cursive).

Podemos também modificar o tamanho da fonte utilizada para obter outro tipo de estética.

Temos a possibilidade de aumentar (caso mostrado aqui) ou diminuir o espaçamento entre as linhas, deixando os textos com uma melhor apresentação ou mesmo auxiliando na leitura para pessoas certas limitações de visão.

Outra possibilidade e alterar o alinhamento dos textos. Este parágrafo está alinhado à direita, os títulos das seções estão alinhados à esquerda, já o título principal da página está alinhado com o centro da página.

As cores das letras também podem ser modificadas, seja no texto inteiro ou em apenas um parágrafo.

Marcadores e identificadores CSS

Para fazer as mudanças nos texto foram utilizados marcadores e identificadores CSS, assim sabemos exatamente onde iremos implementar uma modificação na página. Os marcadores utilizados até agora foram o "class=""" e id="". O primeiro pode ser utilizado para identificar mais de uma tag com o mesmo nome, por exemplo class="Nome_da_classe". Já a segunda deve ser única para cada tag.

Outra forma é utilizar diretamente o nome da tag em nosso CSS e fazedr as mudanças desejadas toda vez que a tag for utilizada.

Mesmo que não apliquemos nenhum estilo alguns estilos são aplicados pelos navegadores que usamos (Chrome, Firefox, etc.), é possível mudar isso utilizando um marcador especial para nosso CSS. O marcador "*" indicará quais estilos usaremos por padrão, eliminando os estilos do navegador.

O mundo como caixas e contêineres

Podemos imaginar toda página web como uma caixa ou conjuntos de caixas/contêires. Cada seção desta página, bem como o cabeçalho no alto da página podem ser vistos como pequenas caixas dentro de uma grande caixa (a página inteira).

Para uma ilustração veja esta seção. Colocamos uma borda com expessura e com cor vermelha para delimitar a caixa desta seção.
